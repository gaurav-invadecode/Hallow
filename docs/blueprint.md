# **App Name**: InvadeCode Teams Lite

## Core Features:

- Direct Messaging: Enable one-to-one text based communication.
- Group Chat: Facilitate text based group conversations.
- File Sharing: Allow users to share files within chats.
- Basic Search: Provide search functionality to find specific messages.
- Smart Compose (replies): AI-powered suggestions for quick replies, with reasoning to adapt to context, making reply writing more efficient as a tool.
- Notifications: Notify users of new messages.

## Style Guidelines:

- Primary color: InvadeCode Indigo (#635BFF), reflecting the InvadeCode brand, providing a modern feel.
- Background color: Light gray (#F2F2F7), a desaturated tone of the primary color providing a clean and modern backdrop.
- Accent color: Analogous purple (#8A7FF7), different in brightness and saturation, providing contrast without straying too far from the primary.
- Font: 'Inter', a sans-serif font, for both headlines and body text due to its modern and neutral look.
- Clean and minimal layout, to feel faster and clearer than existing solutions with an intutive infomation hierarchy.
